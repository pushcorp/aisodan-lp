AI相談.comのLINEの使い方

プランの加入方法

LINE月額プランの加入方法を解説します。

  1. LINEのチャット画面を開く
  2. 画面下の「プラン」をタップ。
  3. 有料プラン「はじめる」をタップ。決済画面が開きます。
  4. 決済情報を入力してください。ご利用可能な決済方法は、クレジットカード、デビットカード、Apple Pay(iPhoneのみ利用可能)、Google Pay(アンドロイド端末のみ利用可能)の4種類です。

プランの解約方法

LINE月額プランの解約方法を解説します。

  1. LINEのチャット画面を開く
  2. 画面下の「設定・情報」をタップ。
  3. 設定・情報「解約する」をタップ。
  4. 本当に解約しますか？→「解約する」をタップ。

クレジットカード変更方法

LINE月額プランのクレジットカード変更方法を解説します。

  1. LINEのチャット画面を開く
  2. 画面下の「設定・情報」をタップ。
  3. 「お支払い情報」をタップ。決済情報ページのリンクが届くので、そのリンクをタップする。
  4. 支払い方法の欄でクレジットカード情報を追加し、「デフォルトの支払い方法」に設定する。

再申し込み方法

LINE月額プランの再申し込み方法を解説します。月額プランを一度解約された方が、もう一度月額プランに加入する際の手順です。

  1. LINEのチャット画面を開く
  2. 画面下の「設定・情報」をタップ。
  3. 「再申し込みする」をタップ。
  4. チャットの指示の通り、プランを再度開始します。
